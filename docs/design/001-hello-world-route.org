* Summary

Add a simple "hello world" GET endpoint that returns a greeting message.
This serves as the foundational route to verify the API is working end-to-end.

* Motivation

Every API needs a basic smoke-test endpoint beyond a health check. A hello
world route provides a minimal, human-friendly response that confirms
routing, serialization, and the request/response cycle are functioning.

* Design

A single GET endpoint at =/hello= that returns a JSON greeting.

** Response Schema

#+begin_src json
{
  "message": "Hello, world!"
}
#+end_src

** Route Location

Add to =app/routers/hello.py= and register in =app/main.py=.
